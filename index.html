¡¡¡Bienvenido a la API de Animalitos Sabios!!! 🐈

¿Qué ofrece?

Imágenes de gatitos, perritos y ajolotitos aleatorias, con frases bonitas aleatorias...

Consume desde cualquier servicio que estés creando; aplicaciones web, móviles, de escritorio, bots, ¡LO QUE SEA!

Si quieres integrar en tu apliación frases bonitas con animalitos bonitos, sólo llama a esta API...

¿Cómo usarla?

Consume como cualquier API REST, desde el link de esta página, según lo que quieras solicitar.

Haz petición "http" a alguno de los siguientes links, según prefieras:

  • 🐱 Para frases de gatitos famosos: GET /michi
  • 🐶 Para frases de perritos motivadores: GET /lomito
  • 💗 Para frases de ajolotitos sabios: GET /ajolotitos
